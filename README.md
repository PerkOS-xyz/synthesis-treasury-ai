# TreasuryAI - Autonomous Corporate Finance System

**Professional Next.js application with App Router, shadcn/ui, and Base Mainnet integration**

## 🚀 Live Demo

**Deployed on Base Mainnet:** https://treasuryai-perkos-demo.netlify.app

## 🏗️ Architecture

This is a modern Next.js 14 application built for the **Synthesis Hackathon 2026** demonstrating enterprise-grade autonomous treasury management.

### Tech Stack

- **Frontend:** Next.js 14 with App Router
- **UI Components:** shadcn/ui + Radix UI
- **Styling:** Tailwind CSS with custom PerkOS theme  
- **Web3:** RainbowKit + Wagmi for wallet integration
- **Blockchain:** Base Mainnet (Chain ID: 8453)
- **TypeScript:** Full type safety
- **Deployment:** Netlify with static export

### Project Structure

```
├── app/                    # Next.js App Router
│   ├── layout.tsx         # Root layout with Web3 providers
│   ├── page.tsx           # Main treasury interface
│   ├── providers.tsx      # RainbowKit + Wagmi setup
│   └── globals.css        # Global styles + shadcn theme
├── components/ui/          # shadcn/ui components
├── contracts/              # Smart contract source files
│   └── TreasuryVault.sol
├── lib/                    # Utilities and configuration
└── Configuration files
```

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- MetaMask or compatible Web3 wallet

### Installation

```bash
# Clone the repository
git clone https://github.com/PerkOS-xyz/synthesis-treasury-ai.git
cd synthesis-treasury-ai

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
